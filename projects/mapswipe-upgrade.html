About

MapSwipe allows anyone with a smartphone to help map remote places and help with humanitarian action. A joint project between MSF, the Red Cross, and HOTOSM, MapSwipe has been used in multiple countries and has thousands of users.

Project

MapSwipe was created 3 years ago. Unfortunately, despite has usage, the App has not been updated. A few volunteers have helped make the code open-source, and we are now looking for volunteers to help upgrade the app and bring it to the new age.

Impact

This project will enable development of new features on MapSwipe.

Skills

We are looking for one-two engineers with strong experience building mobile apps using react-native.

Time Commitment

We expect the first project - upgrade - to take around 10 hours to complete. The timeline is set to 2 weeks.
